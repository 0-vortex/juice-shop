Tutorial Mode

Gradually unlocking tutorials and the entire Score Board

Challenge Tags

  • Shenanigans marks challenges which are not considered serious
  • Contraption indicates a slightly forced or crafted scenario
  • OSINT challenges require Internet research or social stalking
  • Good Practice challenges promote good (security) practices
  • Danger Zone challenges are off on Docker/Heroku by default
  • Good for Demos challenges are suitable for live demos/trainings
  • Prerequisites need to be solved before one or more challenges
  • Brute Force challenges require automation of some security tool
  • Tutorial challenges exist to assist newcomers to Juice Shop
  • Code Analysis of the app or 3rd party coce can be helpful here
